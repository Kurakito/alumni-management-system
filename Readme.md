# Alumni Management System

## Git Flow
1. Create a Branch in Github
2. On Git GUI Click Fetch
3. Create Branch > Match Tracking Branch name > (Select ung branch na ginawa sa gh)
4. Finish your task
5. Stage Changed
6. Commit Changed
7. Push
8. Create a Pull Request
